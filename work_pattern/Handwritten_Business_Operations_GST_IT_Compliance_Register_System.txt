1) ONE-PAGE DAILY FLOW SOP (सुबह-शाम)

MORNING (Start of Day)
[ ] A12 Cash Book: Opening cash count = physical cash (sign)
[ ] A11 Bank Book: Check bank/UPI messages; note pending credits/debits
[ ] A14/A15 Outstanding: Today’s collection targets + vendor payments due
[ ] A16/A18 Plan: Pending Sales Orders / Dispatch / Delivery Challans
[ ] A5 Issue quotes/PI (if needed) → log
[ ] A6 Raise Sales Invoice (or A18 DC first) with GST logic:
    - POS State, Supply Type (L/I/E/SEZ), GST Type (CGST+SGST/IGST), HSN/SAC
[ ] A22 Update stock OUT (if goods) + link Invoice/DC
[ ] A7 Record Purchase Invoices received (mark ITC Code E/B/R/N, RCM flag, ITC Claim Month)
[ ] A22 Update stock IN (if goods) + link purchase invoice/GRN
[ ] A8 Record Expense bills (GST/Non-GST + Cost Center/Project + ITC Code)
[ ] C2 Check TDS/TCS triggers BEFORE payment/credit; mark Section Code (if applicable)

EVENING (End of Day)
[ ] A10 Record receipts/collections; issue receipt no; update A14 ageing
[ ] A9 Record payments; if TDS deducted, ensure C2 entry exists
[ ] A13 Petty cash vouchers attached + approvals
[ ] A12 Cash Book: Closing cash count (verify) + sign
[ ] A11 Bank Book: Post all bank/UPI entries done today
[ ] A25 GL Posting Index: Post today’s docs to ledger folios (minimum: index updated)
[ ] A28 Numbering Control: Check for missing/cancelled doc numbers; log exceptions
[ ] A29 Filing Index: File originals in physical file; scan/photo to digital folder
[ ] Exception Log (A30): Note any control failures and corrective action
